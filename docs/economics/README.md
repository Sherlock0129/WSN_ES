### 2. [AOI的经济学解释](./AOI的经济学解释.md)

**概述**：从经济学角度深入分析Age of Information (AOI，信息年龄)在系统中的核心作用和经济含义。

**主要内容**：
- AOI的基本经济学定义（信息价值的时间衰减）
- AOI的经济学理论框架（信息价值衰减理论、延迟成本理论、时间价值理论）
- AOI在系统中的作用机制（延迟上报、机会路由、连续传输）
- AOI的经济学优化策略
- AOI的经济学模型
- AOI的经济学分析框架

**核心观点**：
1. **AOI = 信息价值的时间衰减**：类似资产折旧或商品保质期
2. **延迟成本与收益权衡**：延迟上报节省能耗，但增加AOI
3. **时间价值理论**：AOI体现信息的时间价值，类似折现理论
4. **规模经济效应**：信息量累积具有规模经济效应，但需要权衡时间成本
5. **信息市场机制**：AOI影响信息市场的供需关系和价格机制

**适用场景**：
- 深入理解AOI的经济学含义
- 优化延迟上报策略
- 优化信息感知路由
- 优化传输时长选择
- 设计信息时效性定价机制

### 5. [AOI与EAoI的区别与联系](./AOI与EAoI的区别与联系.md)

**概述**：解释AOI（Age of Information，信息年龄）和EAoI（Energy Age of Information，能量信息新鲜度）的区别与联系。

**主要内容**：
- AOI的定义和实现（已实现）
- EAoI的定义和理论模型（未实现）
- AOI与EAoI的对比分析
- 当前项目中的应用情况
- EAoI的实现建议

**核心发现**：
1. **AOI**：信息到达物理中心后的时间，已实现并广泛应用
2. **EAoI**：结合节点能量状态的信息新鲜度，未实现
3. **区别**：AOI关注信息的时间衰减，EAoI关注能量-信息协同优化
4. **当前项目**：虽然未实现EAoI，但存在部分能量-信息关联机制

**适用场景**：
- 理解AOI和EAoI的区别
- 了解当前项目的实现情况
- 评估是否需要实现EAoI
